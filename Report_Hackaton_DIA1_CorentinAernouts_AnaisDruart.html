We also had the idea to use some unsupervised algorithms to find out if the user will like the music or not. We first thought of k-means :

set.seed(4004)
km=kmeans(training_set[,c(-15,-16)],centers=2,iter.max = 20)

But the result was with the margin of success of 55.8062741 % which is not good enough to be compared with Random Forest. We didnt send any test using this model.

Hierarchical clustering

Another idea was to use hierchical clustering.

set.seed(4004)
